1. Create and Activate a Virtual Environment
On Windows:
  python -m venv venv
  venv\Scripts\activate

On macOS/Linux:
  python3 -m venv venv
  source venv/bin/activate
  
2. Install Dependencies
Install the required Python packages using pip:
  pip install -r [requirements.txt](http://_vscodecontentref_/1)

3. Apply Migrations
Run the following commands to set up the database:
  python [manage.py](http://_vscodecontentref_/2) makemigrations
  python [manage.py](http://_vscodecontentref_/3) migrate

4. Create a Superuser
To access the Django admin panel, create a superuser:
  python [manage.py](http://_vscodecontentref_/4) createsuperuser

5. Run the Development Server
Start the Django development server:
  python [manage.py](http://_vscodecontentref_/5) runserver

6. Access the Application
Open your browser and go to:
  http://127.0.0.1:8000/

Python Commands for Running the Project
If you download this project and want to run it on your system, follow these commands:

Clone the repository:
    git clone https://github.com/your-username/django-employees.git

Navigate to the project directory:
    cd django-employees

Create and activate a virtual environment:
    python -m venv venv
    venv\Scripts\activate  # On Windows
    source venv/bin/activate  # On macOS/Linux

Install dependencies:
  pip install -r requirements.txt
  
Apply migrations:
  python manage.py makemigrations
  python manage.py migrate

Run the development server:
  python manage.py runserver
